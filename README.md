
# Employee Management System

A web application built with Java and Spring Boot to manage employee records.

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- HTML, CSS, JavaScript

## Features
- Add new employee
- View all employees
- Update employee details
- Delete employee

## How to Run
1. Clone this repository
2. Open in Eclipse or IntelliJ
3. Run as Spring Boot Application
4. Open browser and go to http://localhost:8080
